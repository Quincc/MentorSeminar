Инструкция:
	1	Откройте файл .bashrc в домашней директории пользователя:bash  nano ~/.bashrc 
	2	Добавьте в конце файла строку для добавления директории в PATH:bash  export PATH="$PATH:/path/to/new/dir"  
	3	Сохраните изменения и закройте редактор.

Для применения изменений без закрытия терминала выполните команду:

source ~/.bashrc
